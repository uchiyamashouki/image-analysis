# image-analysis

投球動画（単一カメラ）から、腰の移動軌道とステップ幅を**ブラウザ上で簡易解析**するツールです。

> ⚠️ 本ツールは「比較・傾向把握」向けです。厳密な3Dモーションキャプチャ用途ではありません。

## 特徴

- 1ページ内で解析ワークスペースを複製し、複数動画を並べて比較可能
- MediaPipe Pose による姿勢推定（ブラウザ内で実行）
- 以下のサマリー指標を表示
  - 最大ステップ幅
  - 腰の投球方向移動範囲
  - 腰の腹背方向移動範囲
  - 最大腰速度
  - 推定下肢長
  - ステップ幅 / 推定下肢長 などの比率
  - 推定カメラ角度
- 出力機能
  - CSV ダウンロード
  - 軌道 PNG ダウンロード
  - 注釈付き動画の書き出し

## 動作環境

- 推奨ブラウザ: Google Chrome / Microsoft Edge（最新）
- ネットワーク接続: 初回の MediaPipe モデル・WASM 取得時に必要
